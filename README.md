Este é um projeto de **Jogo da Velha (Tic-Tac-Toe)** desenvolvido em Python, com foco em interatividade e experiência do usuário.

🚀 Funcionalidades

### 🔊 Sistema de Áudio
- Música de fundo  
- Sons de clique  
- Efeitos sonoros de:
  - Vitória 🏆  
  - Derrota ❌  
  - Empate 🤝  
- Opção de **ligar/desligar música e efeitos separadamente**

### 🧠 Níveis de Dificuldade
- Fácil  
- Médio  
- Difícil (IA avançada)  
  - Utiliza um algoritmo treinado com IA  
  - **Impossível de vencer** (no máximo empatar)


## ▶️ Como Executar

1. Certifique-se de ter o **Python 3.x** instalado  
2. Clone este repositório ou baixe os arquivos  
3. Execute o arquivo principal:

```bash
python main.py
